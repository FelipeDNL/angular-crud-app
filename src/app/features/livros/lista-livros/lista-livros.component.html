Sobre Este Projeto

Fiz este projeto para praticar minhas habilidades usando Angular e TypeScript. O projeto é um C.R.U.D (Create, Read, Update, Delete) simples que, como o nome já diz (em inglês), permite adicionar, ler, editar e excluir livros de uma lista. A lista é exibida em uma tabela, onde cada livro é listado com seu ID, título, autor, editora e ano de publicação, sendo essa lista armazenada em um simples array sem conexão com um banco de dados e portanto sem persistencia de dados (ao atualizar a página os dados são perdidos). Para um projeto com integração com banco de dados clique aqui.

Mais Detalhes!

Utilizei SCSS para estilizar e deixar o site visualmente mais interesssante, utlizando uma estética de jornal antigo que mais ou menos encaixa na temática de livros (talvez nem tanto mas é melhor que uma página chata).
